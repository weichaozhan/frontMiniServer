# 说明

## 安装依赖

安装 npm 淘宝镜像

```
npm install -g cnpm --registry=https://registry.npm.taobao.org
```
```
cnpm install
```

## 命令

### 启动本地前端开发与 mock 服务
```bash
npm run start
```

### 本地前端开发服务
```bash
# 端口 3000
npm run dev
```

### 启动本地 mock 服务
```bash
# 端口 9091
npm run mock
```

### 打包编译
```bash
npm run build
```
